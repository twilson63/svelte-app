# Svelte App

Create Svelte App using parcel bundler

## Developer Usage

> [appName] should be replaced with the name of your app

```
npm install degit -g
degit twilson63/svelte-app [appName]
cd [appName]
npm install
npm run dev
```

## Advanced usage

> Install pnpm

```
npm install pnpm degit -g
degit twilson63/svelte-app [appName]
cd [appName]
pnpm install
npm run dev
```

## Deploy

```
npm run build
now dist --static
```
